# Versus Clone (Week 14 網頁設計作業)

這是一個模仿 [Versus](https://versus.com/) 比較網站的前端實作專案。本專案展示了現代化的響應式網頁設計，包含動態視窗切換、互動式模態框(Modals)以及精美的 UI 動畫。

##  功能特色

- **多視圖切換**: 支援 Landing Page、產品網格列表 (Grid) 與 詳細比較頁面 (Comparison) 的無縫切換。
- **互動式卡片**: 
  - 點擊任一產品或文章卡片，即會彈出詳細資訊視窗。
  - 包含圖片、標題與詳細描述。
- **模擬登入系統**:
  - 點擊右上角使用者頭像可喚出登入視窗。
  - 包含美觀的輸入表單與互動按鈕。
- **響應式與視覺設計**:
  - 使用 CSS Grid 與 Flexbox 進行排版。
  - 精緻的漸層背景與玻璃擬態 (Glassmorphism) 效果。
  - 移除干擾閱讀的廣告區塊。

##  技術棧

- **HTML5**: 語意化標籤結構。
- **CSS3**: 使用 CSS Variables 定義主題色，無依賴任何外部框架 (如 Bootstrap)。
- **JavaScript (Vanilla)**: 處理 DOM 操作、以及 Modal 開關邏輯。

##  如何執行

1. Clone 或下載此專案到本地。
2. 直接點擊 `index.html` 在瀏覽器中開啟即可預覽。

##  檔案結構

- `index.html`: 主要頁面結構。
- `styles.css`: 所有樣式定義 (包含 Modal, Grid, Header 等)。
- `script.js`: 頁面互動邏輯 (Modal 控制, 視圖切換)。
